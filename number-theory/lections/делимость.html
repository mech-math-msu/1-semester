Делимость

Теоремы о делении с остатком

Определение: Пусть \(a \in \mathbb{Z}, b \in \mathbb{Z}, a \ne 0\). Если \(\exists c \in \mathbb{Z}\), такое что \(b = ac\), то \(a\,\,|\,\,b\) (\(a\) делит \(b\))

Простейшие свойства

\(\forall a, b, c \in \mathbb{Z}\)

  1. \(1\,\,|\,\,a\)
  2. Если \(a \ne 0\), то \(a\,\,|\,\,a\)
  3. Если \(a\,\,|\,\,b\), то \(a\,\,|\,\,bc\)
  4. Если \(a\,\,|\,\,b\) и \(b\,\,|\,\,с\), то \(a\,\,|\,\,c\)
  5. Если \(a\,\,|\,\,b\) и \(a\,\,|\,\,с\), то \(a\,\,|\,\,(b \pm c)\)
  6. Если \(a\,\,|\,\,b\), то \(|a| \le |b|\)

Простейшие функции

Определение: ((нижняя) целая часть числа) \([x] = \max \{z \in \mathbb{Z}: z \le x\} = \lceil x \rceil\)

\([-4.5] = -5\)
\([5.6] = [5]\)

Определение: (дробная часть числа) \(\{x\} = x - [x]\)

Определение: (верхняя целая часть числа) \(\lfloor x \rfloor = \min \{z \in \mathbb{Z}: z \ge x\}\)

\(\lfloor -4.5 \rfloor = -4\)
\(\lfloor 5.5 \rfloor = [6]\)

Определение: (расстояние до ближайшего целого) \(\displaystyle ||x|| = \min_{a \in \mathbb{Z}} |x - a|\)

Упражнение: \(||x|| = \min (\{x\}, 1 - \{x\})\)

Упражнение: \(||x + y|| \le ||x|| + ||y||\)

Теорема: (обычное деление с остатком) \[\forall a \in \mathbb{Z},\, b \in \mathbb{N}\,\,\,\, \exists !\,\, q, r \in \mathbb{Z}: a = bq + r,\,\, 0 \le r < b\]

Доказательство:

Существование: 1. Если \(a\,\,|\,\,b\), то \(q = \frac{a}{b} \in \mathbb{Z},\,\,\,\,r = 0\). 2. Иначе рассмотрим множество \[M=\{z \in \mathbb{N}: z = a - bq, q \in \mathbb{Z}\}\] Докажем, что \(M \ne \varnothing\) рассмотрим \(q_0 = -|a| - 1\) \[z = a - bq_0 = a + b(|a| + 1) \ge a + |a| + 1 \ge 1 \Rightarrow z \in \mathbb{N} \Rightarrow z \in M\] Следовательно, по аксиоме Архимеда \(\exists r = \min M = a - bq\). \[r_1 = r - b = a - b(q + 1), \,\,\,\, b \in \mathbb{N} \Rightarrow r_1 < r \Rightarrow r_1 \not\in M \Rightarrow \] \[r_1 \le 0, \,\,\,\, r \ne 0 \Rightarrow r_1 < 0 \Rightarrow 0 \le r < b \,\,\,\,\blacksquare\]

Единственность:

Пусть: \[ a = bq_0 + r_0\] \[a = bq_1 + r_1 \Rightarrow\] \[0 = b(q_1 - q_0) + (r_1 - r_0) \Rightarrow b\,\,|\,\,(r_0 - r_1) \Rightarrow b \le |r_0 - r_1|\] \[0 \le r_0, r_1 \le b - 1 \Rightarrow |r_0 - r_1| \le b \Rightarrow r_0 = r_1, \,\,\, q_0 = q_1 \,\,\,\,\blacksquare\]

НОД

Определение: (НОД) Пусть \(a, b \in \mathbb{Z},\,\,\,\,\mathcal{D}_{a, b}= \{z \in \mathbb{N}: z\,\,|\,\,a, z\,\,|\,\,b\}\). Тогда \[\mathbf{НОД}(a, b) = \max \mathcal{D}\]

Утверждение: (корректность определения) Доказательство: очев.

Лемма: \[(a, b) = (a, a \pm b)\]

Доказательство: \[\begin{cases} z\,\,|\,\,a \\ z\,\,|\,\,b \end{cases} \Rightarrow z\,\,|\,\,b \pm a \Rightarrow \mathcal{D}_{a, b} \subset \mathcal{D}_{a, b \pm a}\] \[\begin{cases} z\,\,|\,\,a \\ z\,\,|\,\,b + a \end{cases} \Rightarrow z\,\,|\,\,b \Rightarrow \mathcal{D}_{a, b+a} \subset \mathcal{D}_{a, b}\,\,\,\,\blacksquare\]

Алгоритм Евклида

\[a \in \mathbb{Z}, b \in \mathbb{N}\] \[a = bq_0 + r_1, \,\,\,\, 0 \le r_1 < b\] \[b = r_1q_1 + r_2, \,\,\,\, 0 \le r_2 < r_1\] \[r_0 = r_2q_2 + r_3, \,\,\,\, 0 \le r_3 \le r_2\] \[\vdots\] \[r_{j - 1} = r_{j}q_{j} + r_{j + 1}, \,\,\,\, 0 \le r_{j + 1} \le r_j\] \[\vdots\] \[r_{n - 2} = r_{n - 1}q_{n - 1} + r_{n}, \,\,\,\, 0 \le r_{n} \le r_{n - 1}\] \[r_{n - 1} = r_{n}q_{n} + r_{n + 1}, \,\,\,\, r_{n + 1} = 0\] \[\exists n \in \mathbb{N}: r_{n + 1} = 0\]

Утверждение: \(\mathbf{НОД} (a, b) = r_n\)

Доказательство: \(\forall j \le n (r_{j - 1}, r_j) = (r_j, r_{j + 1}) \,\,\,\,\blacksquare\)

Теорема: (о линейном представлении НОД) Пусть \(a, b \in \mathbb{Z}, \,\,\,\,\,\, d = (a, b)\). Тогда \[\exists x, y \in \mathbb{Z}: d = ax + by\]

Доказательство: Обратный ход алгоритма Евклида \(\,\,\,\,\blacksquare\).

Лемма: (важная) \[\begin{cases} a\,\,|\,\,bc \\ (a, b) = 1 \end{cases} \Rightarrow a\,\,|\,\,c\] Доказательство: \[(a, b) = 1 \Rightarrow \exists x, y \in \mathbb{Z}: 1 = ax + by\] \[acx + bcy = c\] \[\begin{cases} a\,\,|\,\,acx \\ a\,\,|\,\,bcy \end{cases} \Rightarrow a\,\,|\,\,c \,\,\,\,\blacksquare\]

НОК и НОД нескольких чисел

Определение: \(z \in \mathbb{Z}\) - общее кратное чисел \(a_1, \ldots, a_n\), если \(a_j \,\,|\,\,z \forall j\)

Определение: \(\mathbf{НОК} (a_1, \ldots, a_n) = \min \{z \in \mathbb{N} : z\) общее кратное чисел \(a_1, \ldots, a_n \}\)

Теорема: Если \(z_0 - \mathbf{НОК} (a_1, \ldots, a_n)\), то \(z_0 \,\,|\,\,z\)

Доказательство:

Определение: \(\mathbf{НОД} (a_1, \ldots, a_n) = \max \{z \in \mathbb{N}: z \,\,|\,\,a_j \,\,\,\,\,\, \forall j = 1, \ldots, n\}\)

Теорема: Пусть \(d\) - общий делитель \(a_1, \ldots, a_n\). Тогда \(d \,\,|\,\,\mathbf{НОД} (a_1, \ldots, a_n)\)
